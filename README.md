# C2

Простой браузерный калькулятор в одном HTML-файле.

## Запуск

Откройте файл `index.html` в любом современном браузере.

## Возможности

- сложение, вычитание, умножение и деление;
- ввод мышью и с клавиатуры;
- очистка, удаление последнего символа и обработка ошибок.
